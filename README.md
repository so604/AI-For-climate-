#  AI Climate & Disaster Prediction System

This project is a Machine Learning-based system that analyzes historical climate data and predicts future temperature trends. It also provides climate risk analysis based on predicted temperatures.

# Project Overview

- Dataset: Global Land Temperatures by City (Kaggle)
- Model: Linear Regression
- Objective: Predict future temperature trends and analyze climate risk

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

# Machine Learning Model

- Algorithm: Linear Regression
- Input Feature: Year
- Target: Average Temperature
- Model Performance: R² Score ≈ 0.81

# Features

 Data Cleaning  
 Trend Visualization  
 Future Temperature Prediction  
 Climate Risk Analysis  
 Graphical Insights  

# Future Predictions

- 2030 → ~20.95°C  
- 2040 → ~21.33°C  
- 2050 → ~21.70°C  


# Climate Risk Analysis

- Moderate to High Heatwave Risk predicted in future years



# Conclusion

This project helps understand long-term climate change trends using Machine Learning and can be extended for real-world disaster prediction systems.
